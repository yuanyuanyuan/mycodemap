## 评分标准

| ID | 检查点 | 分值 | 验证方法 | 自动/手动 |
|----|--------|------|----------|----------|
| L1-1 | [检查点1] | 25 | [验证命令] | 自动 |
| L2-1 | [检查点2] | 25 | [验证命令] | 自动 |
| L3-1 | [检查点3] | 25 | [验证命令] | 自动 |
| L4-1 | [负面检查] | 25 | [验证命令] | 自动 |

> 以上分值可按任务调整，但**总分必须为 100**。

### 评分等级
- **通过 (Pass)**: >= 70 分
- **优秀 (Excellent)**: >= 90 分
- **失败 (Fail)**: < 70 分

### 验证环境
- Node.js 版本: >= 18
- 测试框架: Vitest
- 执行命令: `pnpm test`

### 验证命令示例
```bash
# 自动验证
pnpm test .tasks/{task-name}/EVAL.ts

# 手动验证
[验证步骤说明]
```

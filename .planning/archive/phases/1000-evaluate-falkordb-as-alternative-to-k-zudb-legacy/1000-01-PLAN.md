## Task List

### Task 1: Implement ArcadeDBStorage Adapter Core
**ID:** 1000-01-t1
**Status:** pending

**Action:**
Create `src/infrastructure/storage/adapters/ArcadeDBStorage.ts` with full IStorage implementation.

**Key implementation details:**
- Use embedded mode: `new ArcadeDB({ mode: 'embedded', path: config.path })`
- Implement all abstract methods from StorageBase:
  - `doInitialize()`: Setup embedded database, create schema if needed
  - `doClose()`: Clean shutdown of database
  - `saveCodeGraph(graph)`: Batch insert nodes (File, Symbol) and edges (imports, calls)
  - `loadCodeGraph()`: Retrieve full graph structure
  - `querySymbols(filters)`: Cypher query with parameterized filters
  - `getDependencies(symbolId)`: Return direct dependencies via Cypher
  - `getCallChain(symbolId, depth)`: Multi-hop traversal query
  - `getModules()`: List all file nodes

**Cypher patterns to use:**
```cypher
// Create schema
CREATE VERTEX TYPE File IF NOT EXISTS
CREATE VERTEX TYPE Symbol IF NOT EXISTS
CREATE EDGE TYPE imports IF NOT EXISTS FROM File TO File
CREATE EDGE TYPE calls IF NOT EXISTS FROM Symbol TO Symbol

// Batch insert (parameterized)
INSERT INTO File SET path = ?, type = ?, size = ?
INSERT INTO Symbol SET name = ?, type = ?, filePath = ?, line = ?, column = ?
CREATE EDGE calls FROM (SELECT FROM Symbol WHERE id = ?) TO (SELECT FROM Symbol WHERE id = ?)

// Query dependencies
MATCH {type: Symbol, id: $id}<-[:defines]-{type: File}-[:imports]->{type: File}-[:defines]->{type: Symbol} as s
RETURN s.name, s.type, s.filePath

// Call chain traversal
MATCH {type: Symbol, id: $startId}.calls{1,$depth}{type: Symbol} as chain
RETURN chain
```

**Files to create/modify:**
- `src/infrastructure/storage/adapters/ArcadeDBStorage.ts` (new)

**Acceptance criteria:**
- [ ] TypeScript compiles without errors
- [ ] All abstract methods implemented
- [ ] Uses parameterized queries (no string concatenation)
- [ ] Proper error handling with typed errors

### Task 2: Update StorageFactory for ArcadeDB Support
**ID:** 1000-01-t2
**Status:** pending

**Action:**
Extend StorageFactory to recognize 'arcadedb' type and integrate with existing fallback mechanism.

**Implementation:**
```typescript
// In StorageFactory.create()
if (config.type === 'arcadedb') {
  try {
    const { ArcadeDBStorage } = await import('./adapters/ArcadeDBStorage.js');
    return new ArcadeDBStorage(config);
  } catch (error) {
    logger.warn('ArcadeDB initialization failed, falling back to filesystem', { error });
    return new FileSystemStorage(config);
  }
}
```

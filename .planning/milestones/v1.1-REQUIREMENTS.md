# Requirements Archive: v1.1 插件扩展点产品化

**Archived:** 2026-03-24  
**Outcome:** All v1.1 requirements validated

## v1.1 Requirements

### Plugin Surface

- [x] `PLG-01`: 用户可以在正式 `CodemapConfig` 中声明内置插件开关
- [x] `PLG-02`: 用户可以配置本地插件目录或显式插件名列表，而不是手改源码
- [x] `PLG-03`: 非法插件配置会在 CLI / 配置加载阶段被校验并给出清晰错误

### Runtime Integration

- [x] `PLG-04`: `generate` 主流程会加载并运行已配置插件
- [x] `PLG-05`: 分析主路径能够暴露已加载插件产生的结果或元数据，而不是只停留在孤立单元测试
- [x] `PLG-06`: 插件加载 / 初始化失败时会产生结构化 warning 或可诊断输出，而不是静默失效

### Documentation & Validation

- [x] `DOC-03`: `README.md`、`AI_GUIDE.md`、`docs/ai-guide/*` 明确说明插件配置、边界与示例
- [x] `DOC-04`: docs guardrail 与相关测试能阻止插件文档和实现再次漂移
- [x] `VAL-01`: 至少一条 built-in plugin 场景与一条用户插件场景具有端到端验证证据

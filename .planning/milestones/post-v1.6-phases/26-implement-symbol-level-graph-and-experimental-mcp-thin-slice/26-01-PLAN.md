<objective>
完成 Phase 26 的首个最小基础设施切片：让 `generate --symbol-level` 能把 `smart-parser` 的 symbol-level 调用真相落到 CodeGraph 和 SQLite 中，并把 `sqlite` backend 从 CLI config 路径正式打通。

Output: 扩展后的 graph/storage contract、可用的 `sqlite` config path、`generate --symbol-level` materializer、focused regression tests。
</objective>

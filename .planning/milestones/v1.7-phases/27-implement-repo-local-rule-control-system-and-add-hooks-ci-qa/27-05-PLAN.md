- Threat: helper 返回全部规则，subagent prompt 被无关上下文淹没，反而更容易漏掉真正相关规则
  - Severity: high
  - Mitigation: helper 只允许返回与输入文件匹配的 1-2 条规则与对应 verify commands
- Threat: advisory hook 在 disabled config 下仍然输出提醒，导致误报和噪声
  - Severity: medium
  - Mitigation: hook 首先读取 `.claude/rule-system.config.json` 的 `enabled` 与 `soft_gate.change_analyzer`，不满足即静默退出
</threat_model>

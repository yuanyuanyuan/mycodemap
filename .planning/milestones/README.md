# Archived Milestones

> This directory stores **historical snapshot** artifacts captured at milestone closeout.
> Current active planning truth always lives in the root `.planning/PROJECT.md`, `.planning/ROADMAP.md`, `.planning/REQUIREMENTS.md`, and `.planning/STATE.md`.

- `*-ROADMAP.md` / `*-REQUIREMENTS.md` / `*-MILESTONE-AUDIT.md`: archive snapshots captured at closeout
- `*-phases/`: archived phase artifacts for that shipped milestone
- These files are read-only historical context unless a later milestone explicitly audits or repairs their identity labels

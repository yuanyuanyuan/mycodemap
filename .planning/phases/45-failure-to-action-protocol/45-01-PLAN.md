# Phase 45 Plan 01: ActionableError Type System & Global Safety Flags

## Objective

Extend the existing `StructuredError` type with actionable fields (`attempted`, `rootCause`, `confidence`, `nextCommand`, `causes`), create a canonical error code registry with prefix classification, upgrade `normalizeError` to auto-assign codes and remediation for common Node.js error patterns, and register `--apply-suggestion` and `--wasm-fallback` as global commander flags.

## must_haves

- [ ] `ActionableError` interface extends `StructuredError` with `attempted`, `rootCause`, `remediationPlan`, `confidence`, `nextCommand`, `causes` fields
- [ ] Error code registry with `DEP_*`, `CFG_*`, `RUN_*`, `FS_*` prefix classification
- [ ] `normalizeError` auto-detects `MODULE_NOT_FOUND`, `EACCES`, `ENOENT` and assigns appropriate error codes + remediation
- [ ] `formatError` renders `ActionableError` fields in both JSON and human modes
- [ ] `--apply-suggestion` and `--wasm-fallback` registered as global commander flags
- [ ] Confidence threshold constant (0.8) used for `--apply-suggestion` gating

## truths

- Phase 44 `StructuredError` is the base type — we extend, never replace
- Phase 43 `DiagnosticResult.nextCommand` is `string?` — we match this interface
- Error codes are English kebab-case with prefix (e.g., `DEP_NATIVE_MISSING`) not numeric

```typescript
function normalizeError(error: unknown, attempted?: string): ActionableError {
  if (error instanceof Error) {
    const err = error as ErrorWithCode & { code?: string; syscall?: string; path?: string };
    const detected = detectErrorPattern(err);
    return {
      type: 'error',
      code: detected.code,
      message: err.message,
      attempted: attempted ?? 'unknown operation',
      rootCause: detectRootCause(err),
      remediationPlan: detected.remediation.message,
      confidence: detected.remediation.confidence,
      nextCommand: detected.remediation.nextCommand,
      ...(detected.remediation.nextCommand ? { remediation: detected.remediation.nextCommand } : { remediation: detected.remediation.message }),
      causes: extractCauses(err),
    };
  }

  // Non-Error thrown values
  return {
    type: 'error',
    code: ErrorCodes.RUN_COMMAND_FAILED,
    message: String(error),
    attempted: attempted ?? 'unknown operation',
    rootCause: String(error),
    remediationPlan: ErrorRemediation.RUN_COMMAND_FAILED.message,
    confidence: 0.1,
  };
}
```

- Default → `RUN_COMMAND_FAILED`

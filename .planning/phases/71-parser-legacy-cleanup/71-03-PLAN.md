<task id="1" name="Introduce parser resolver abstraction in Interface layer">
<description>
In `src/interface/types/parser.ts` (or a new file `src/interface/types/parser-resolver.ts`):
1. Define a minimal resolver interface:
   ```ts
   export interface IParserResolver {
     getParserByFile(filePath: string): ILanguageParser | undefined;
     getSupportedLanguages(): LanguageId[];
     initializeParsers(): Promise<void>;
     disposeParsers(): Promise<void>;
   }
   ```
2. Alternatively, if keeping it simpler, add `parserRegistry?: IParserRegistry` to `AnalysisOptions` and let `analyzer.ts` use it when provided, falling back to a lazy default only if absent.
3. The key principle: Core should not KNOW how to construct a registry; it should only USE one.

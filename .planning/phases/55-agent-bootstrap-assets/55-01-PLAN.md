3. **`extractManifestFacts(rootDir: string, profileName?: string): ManifestFacts` function:**

Logic:
- Set `projectType` from `profileName ?? 'generic'` and `projectSource` from the marker file that was detected.
- Try reading `package.json` (existsSync + readFileSync + JSON.parse):
  - If `scripts.test` exists: add `{ category: 'execution', key: 'testCommand', value: scripts.test, source: 'package.json:scripts.test', confidence: 'high' }`
  - If `scripts.build` exists: add `{ category: 'execution', key: 'buildCommand', value: scripts.build, source: 'package.json:scripts.build', confidence: 'high' }`
  - If `scripts.lint` exists: add `{ category: 'execution', key: 'lintCommand', value: scripts.lint, source: 'package.json:scripts.lint', confidence: 'high' }`
  - If `main` exists: add `{ category: 'execution', key: 'entryPoint', value: main, source: 'package.json:main', confidence: 'medium' }`
- Try reading `pyproject.toml` (existsSync + readFileSync + smol-toml parse):
  - Import `parse` from `smol-toml` (already available from Phase 54).
  - If `[project]` section has `name`: add `{ category: 'execution', key: 'projectName', value: name, source: 'pyproject.toml:project.name', confidence: 'high' }`
  - If `[tool.pytest.ini_options]` exists: add `{ category: 'execution', key: 'testCommand', value: 'pytest', source: 'pyproject.toml:tool.pytest', confidence: 'medium' }`
- Try reading `go.mod` (existsSync + readFileSync):
  - First line matching `module (.+)`: add `{ category: 'execution', key: 'moduleName', value: match[1], source: 'go.mod:module', confidence: 'high' }`
- Try reading `Cargo.toml` (existsSync + readFileSync + smol-toml parse):
  - If `[package]` has `name`: add `{ category: 'execution', key: 'packageName', value: name, source: 'Cargo.toml:package.name', confidence: 'high' }`
- Add unknown items for categories not detected:
  - If no `testCommand` item was added: add `{ category: 'execution', key: 'testCommand', status: 'unknown', source: 'not-detected', confidence: 'none' }`
  - If no `buildCommand` item was added: add `{ category: 'execution', key: 'buildCommand', status: 'unknown', source: 'not-detected', confidence: 'none' }`

Error handling: If any file read/parse fails, skip that manifest silently (do not throw). The function should always return a valid ManifestFacts.

Use `import { parse as parseToml } from 'smol-toml'` for TOML parsing.

- `extractManifestFacts` with `go.mod` containing `module example.com/foo` returns `moduleName` item
- `extractManifestFacts` with `Cargo.toml` containing `[package] name = "foo"` returns `packageName` item
- `extractManifestFacts` with `profileName: 'nodejs'` sets `projectType: 'nodejs'`

Use `mkdtempSync` + `writeFileSync` for temp dir fixtures. Clean up in `afterEach`.

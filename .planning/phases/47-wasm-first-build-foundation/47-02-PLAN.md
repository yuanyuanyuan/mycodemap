# Plan 47-02: Optional Dependencies and Install-Time Configuration

## Objective
Make `web-tree-sitter` and `sql.js` optional dependencies so `npm install codemap` succeeds even when native compilation fails. Document the WASM-first install experience.

## Background
Currently `tree-sitter` and `better-sqlite3` are hard dependencies with native bindings. When users install on systems without `python`, `make`, `gcc`, the install fails. By making WASM fallbacks optional dependencies, npm will try to install them but not fail the entire install if they fail. Combined with the wrappers from Plan 47-01, the CLI will gracefully degrade to WASM.

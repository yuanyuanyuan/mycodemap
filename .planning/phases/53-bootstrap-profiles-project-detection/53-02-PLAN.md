Purpose: Without integration, detection and profiles are dead code. This plan makes them operational within the established init asset family pattern.
Output: profile-plan.ts, updated reconciler.ts, updated init.ts, updated init contract.
</objective>

2. **Types:**
   ```typescript
   export interface ProfilePlan {
     assets: InitAsset[];
     writes: FileWriteAction[];
     mergedConfigText?: string;
   }
   interface FileWriteAction {
     targetPath: string;
     content: string;
   }
   ```

3. **`createProfilePlan(rootDir: string, profile: BootstrapProfile | null, scan: InitScan, mode: 'preview' | 'apply'): ProfilePlan`**:
   - If `profile === null`, return `{ assets: [skippedAsset], writes: [] }` where skippedAsset has:
     - `key: 'bootstrap-profile'`, `label: 'bootstrap profile'`, `status: 'skipped'`, `ownership: 'tool-owned'`, `origin: 'project-detection'`
     - `details: ['未检测到项目类型标记；跳过 bootstrap profile 推荐']`
   - Build merged config by starting with `createDefaultCodemapConfigFile()` and overlaying:
     - `include: profile.parser.include`
     - `exclude: [...defaultExclude, ...profile.ignore]` (merge default excludes with profile ignores)
     - `mode: mapAnalysisDepth(profile.analysis_depth)` where mapping is: `shallow -> 'fast'`, `standard -> 'hybrid'`, `deep -> 'smart'`
   - Serialize merged config with `JSON.stringify(mergedConfig, null, 2)`.
   - If `scan.hasCanonicalConfig`:
     - Return asset with `status: 'already-synced'` (D-16), `details: ['检测到已存在的 .mycodemap/config.json；跳过 profile 应用。如需重新检测，请使用 --re-detect']`.
   - If `mode === 'preview'`:
     - Return asset with `status: 'skipped'`, `details: [`推荐 profile: ${profileName}`, '预览模式不会写入配置；使用 --yes 应用']`.
     - Include `mergedConfigText` in the plan for preview display.
   - If `mode === 'apply'`:
     - Push write action: `{ targetPath: scan.paths.canonicalConfigPath, content: mergedConfigText }`.
     - Return asset with `status: 'installed'`, `details: [`应用 profile: ${profileName}`, '已将推荐配置合并到 .mycodemap/config.json']`, `path: scan.paths.canonicalConfigPath`, `hash: fileHash(mergedConfigText)`.

4. **`applyProfilePlan(plan: ProfilePlan): Promise<void>`**:
   - For each write in `plan.writes`: `await mkdir(path.dirname(write.targetPath), { recursive: true }); await writeFile(write.targetPath, write.content, 'utf8');`

5. **Helper `fileHash(text: string): string`**:
   - Use `createHash('sha256').update(text).digest('hex').slice(0, 12)` from `node:crypto` (same as reconciler.ts).

6. **No `process.exit()` in this module.**

<objective>
Make parser orchestration truth single-path: replace hybrid threshold switching with registry-backed resolution, preserve `createParser()` as a compatibility wrapper, and hard-fail deprecated parser modes with actionable remediation.

Purpose: This is the architectural cutover that removes the active fast/smart split and creates one stable boundary for all downstream parser work.
Output: analyzer and parser entry use one orchestrator, `actualMode` is removed, and deprecated mode requests produce structured errors instead of silent routing.
</objective>

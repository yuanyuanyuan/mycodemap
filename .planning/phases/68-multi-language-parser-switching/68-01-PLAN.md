- File extension drives grammar selection; a wrong mapping silently changes parser behavior.
- Native/WASM loader behavior affects whether AST parsing fails closed or degrades unpredictably.
- Shared parser cutover touches the runtime path used by analyzer and benchmark.

| T-68-01-2 | Denial of service | grammar loading | mitigate | Preserve native-first/WASM-fallback behavior and keep explicit total-failure remediation contract in the shared loader. |

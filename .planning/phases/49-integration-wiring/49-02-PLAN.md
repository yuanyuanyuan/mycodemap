# Plan 02: Contract Schema Core Commands

## Objective
将 `doctor`、`benchmark`、`init` 三个核心命令加入 interface contract schema，使 MCP gateway 的 schema-adapter 能自动注册这些命令为 MCP tools。

当前 contract 仅覆盖 `analyze`、`query`、`deps`（3/15+ 命令）。本 plan 按"够用就行"原则，补全最核心的 3 个命令。

## must_haves
- `doctorContract`、`benchmarkContract`、`initContract` 三个 contract 定义文件存在
- `commandContracts` 数组包含这 3 个新 contract
- `codemap --schema` 输出包含 doctor、benchmark、init 的完整契约
- Contract 通过 meta-schema 验证（`validateInterfaceContract` 不抛错）

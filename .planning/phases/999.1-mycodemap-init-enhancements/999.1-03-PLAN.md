<objective>
Add hook reconciliation to `mycodemap init`: ship canonical hook templates, sync them into `.mycodemap/hooks/`, install `.git/hooks/` shims that preserve existing user hooks, append `.mycodemap/logs/` to `.gitignore` exactly once, and report every hook/gitignore outcome in the receipt.

Purpose: Hooks are the most sensitive project-owned boundary; Phase 999.1 must avoid silent overwrite while still making CodeMap guardrails easy to adopt.
Output: Hook helper, packaged templates, shim logic, `.gitignore` handling, and fixture tests.
</objective>

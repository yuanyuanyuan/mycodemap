## 🎯 P1: 函数体关键代码片段 (方案 B)

### 目标
参考 Repomix 的 `--compress` 模式，提取关键代码结构而非完整实现。

### 实现策略

```typescript
// 新增类型
type CodeSnippet = {
  type: 'if' | 'loop' | 'try' | 'switch' | 'early-return';
  lines: string;
  lineStart: number;
  lineEnd: number;
};

// 在 FunctionSignature 中添加
interface FunctionSignature {
  // ... 现有字段
  bodySnippets?: CodeSnippet[];  // 新增
}
```

### 提取规则

```typescript
/**
 * 提取函数体关键片段
 * 
 * 策略: 保留控制流结构，省略实现细节
 * 
 * 示例转换:
 * 
 * 原始代码:
 * async function fetchUserData(userId: string): Promise<User> {
 *   if (!userId) {
 *     throw new Error('User ID required');
 *   }
 *   
 *   try {
 *     const response = await api.get(`/users/${userId}`);
 *     return response.data;
 *   } catch (error) {
 *     logger.error('Failed to fetch user', error);
 *     throw error;
 *   }
 * }
 * 
 * 提取后:
 * async function fetchUserData(userId: string): Promise<User> {
 *   if (!userId) throw new Error(...);
 *   try {
 *     // ⋮ implementation
 *   } catch (error) {
 *     // ⋮ error handling
 *   }
 * }
 */
```

### 实现步骤

1. **类型定义** (15 min)
   - 在 `src/types/index.ts` 添加 `CodeSnippet` 类型
   - 更新 `FunctionSignature` 接口

2. **SmartParser 扩展** (60 min)
   - 新增 `extractBodySnippets()` 方法
   - 处理: if/for/while/do/try-catch/switch
   - 保留条件表达式，用 `// ⋮ implementation` 替换 body

3. **Generator 更新** (30 min)
   - 在 `generateContext()` 中显示 snippets
   - 折叠/展开格式

4. **测试** (30 min)
   - 添加测试用例
   - 验证各种控制流结构

## 🎯 P2: 复杂泛型约束 (基础版)

### 目标
支持 TypeScript 高级泛型特性。

### 实现策略

```typescript
// 扩展现有泛型参数类型
interface GenericParameter {
  name: string;
  constraint?: string;
  default?: string;
  // 新增
  variance?: 'in' | 'out' | 'inout';  // contravariant/covariant/bivariant
}

// 复杂类型支持
interface TypeInfo {
  // ... 现有字段
  conditionalTypes?: ConditionalTypeInfo[];
  mappedTypes?: MappedTypeInfo[];
  templateLiteralTypes?: TemplateLiteralTypeInfo[];
}

interface ConditionalTypeInfo {
  checkType: string;
  extendsType: string;
  trueType: string;
  falseType: string;
}

interface MappedTypeInfo {
  typeParameter: string;
  constraint: string;
  valueType: string;
  modifiers?: ('readonly' | 'optional' | 'nullable')[];
}
```

### 支持场景

```typescript
// 条件类型
T extends U ? X : Y

// 映射类型  
{ [K in keyof T]: T[K] }
{ [K in T as NewKey]: Value }

// 模板字面量类型
`prefix-${T}-suffix`

// 变型注解
type Getter<out T> = () => T;
type Setter<in T> = (value: T) => void;
```

### 实现步骤

1. **类型定义** (20 min)
   - 扩展 `GenericParameter` 和 `TypeInfo`

2. **SmartParser 扩展** (60 min)
   - 增强类型解析逻辑
   - 处理 conditional/mapped/template literal 类型

3. **Generator 更新** (20 min)
   - 格式化显示复杂类型

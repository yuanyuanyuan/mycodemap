# docs/product-specs/

本目录只存放当前仍有效的产品规格，而不是已失效的历史方案。

## 当前状态

- 截至 2026-03-15，历史重构需求草稿已归档到 `docs/archive/product-specs/`。
- 当前活跃产品规格暂为空；若后续需要新规格，请基于当前实现事实重新整理。

## 应放内容

- 用户场景与目标用户
- 功能边界与非功能要求
- 输入输出契约
- 验收标准与不做什么

## 与其他目录的边界

- 实现方案放 `docs/design-docs/`
- 执行计划放 `docs/exec-plans/`
- 开发与验证规则放 `docs/rules/`
- 历史规格草稿放 `docs/archive/product-specs/`
